# FPGA-ADC-AD9248-Acquisition-System
This project shows how to acquire an analog signal using an AD9248 14-bit ADC connected to a Xilinx Spartan-6 XC6SLX9
The FPGA captures the digital samples produced by the ADC and transmits them to a computer through a UART interface. A Python application receives the samples and displays the acquired signal in real time.
